Автоматическая установка

Установка платформы Абрикос на свой сайт из дистрибутива

1. Скачайте дистрибутив, выбрав наиболее подходящую для вашего сайта сборку, на странице скачать
2. Разархивируйте архив в корневой каталог вашего сайта
3. Откройте заглавную страницу вашего сайта и следуйте инструкциям инсталлятора.
