Исходный код

Платформа Абрикос является бесплатной и открытой разработкой, исходный код которой доступен на Github по адресу http://github.com/abricos/

Редактирование страниц сайта ru.abricos.org, abricos.org

Исходный код большинства страниц сайта ru.abricos.org и abricos.org доступны в открытом репозитории на Github - github.com/abricos/abricos.org. Для внесения изменений необходимо зарегистрироваться на Github, форкнуть репозиторий github.com/abricos/abricos.org, внести изменения в исходный код страниц и отправить пулреквестом в основной репозиторий.
