Структура шаблона

Шаблоны в платформе Абрикос располагаются в корневой папке tt. По умолчанию ядро платформы включает шаблон default, в который, настоятельно не рекомендуется вносить изменения, потому как при следующем обновлении платформы все изменения будут затерты. 


  [корневая папка платформы]/
    tt/
      [имя_шаблона]/
        [файл_шаблона1].html
        ...
        [файл_шаблонаN].html
