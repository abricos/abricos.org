Структура модуля

Модули платформы Абрикос располагаются в ее корневой папке modules. 


  [корневая папка платформы]/
    modules/
      [имя_модуля]/
        brick/                   - свободные кирпичи
        content/                 - стартовые кирпичи
        includes/                - php-скрипты
        js/                      - js-компоненты
        language/                - (опционально) языковые фразы 
          [lang_id].php
        tt/                      - (опционально) шаблоны модуля
        module.php               - главный php-скрипт модуля
