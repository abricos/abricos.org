FAQ

Часто задаваемые вопросы по платформе Абрикос

Платформа установлена на сайт, но не проходит авторизация учетной записи администратора (логин: admin, пароль: admin)

Не работает mod_rewrite или запрещен файл .htaccess
Для проверки правильной работы mod_rewrite и .htaccess достаточно зайти на страницу http://domain.tld/about/ созданную по умолчанию в процессе установки движка.


Установил платформу в раздел сайта cms (http://мойсайт.ру/cms/), но она не работает

Платформа Абрикос не работает в разделах сайта, для ее работы она должна находится в корневой директории сайта. Для проверки этого, убедитесь в том, чтобы файл платформы index.php был в корне сайта. Т.е. если папка вашего сайта httpdocs, то index.php должен быть именно в этой папке.
Если нет возможности установить платформу в корень сайта, рекомендуем установить ее в поддомен сайта, например http://cms.мойсайт.ру


При открытие страницы выходит пустое окно

Вероятнее всего возникла ошибка на сервере при генерации страницы и при этом в настройках сервера запрещено отображение ошибок.
Для того, чтобы просмотреть эту ошибку, откройте журнал ошибок вашего сайта - файл error.log.
