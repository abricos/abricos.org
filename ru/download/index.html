Минимальные требования к серверу

Для работы системы необходимы следующие хар-ки вашего сайта:

  • PHP не ниже 5 версии
  • MySql не ниже 4 версии
  • Apache версии 2 и выше с загруженными модулями: mod_rewrite

Установка для версии 0.5.3 и ниже

Для установки движка необходимо сделать следующее:
1) Создайте базу данных на вашем сайте в кодировке utf8 сравнение utf8_general_ci;
2) Извлеките архив в нужный каталог вашего сайта;
3) Переименуйте файл includes/config.new.php в includes/config.php (для сборки WebOS файл config.newwebos.php, если необходимо загружать рабочий стол по корневому адресу сайта);
4) Настройте движок для работы с базой данных MySql в этом файле (config.php): 

	$config['Database']['dbname'] = 'имя_базы_данных';

	$config['Server']['username'] = 'имя_пользователя';
	
	$config['Server']['password'] = 'пароль';


5) Установите права 777 папке cache движка Abricos;
6) Откройте главную страницу вашего сайта в браузере, в этот момент движок установит все необходимые таблицы и откроет главную страницу вашего сайта;
После успешной установки будет создан пользователь admin с правами администратора: 

имя: admin

пароль: admin

После успешной установки платформы Abricos на Ваш сайт, обязательно смените пароль учетной записи admin.
