Модуль «Уведомления на e-mail»

Этот модуль необходим для отправки сообщений на электронную почту пользователей на Вашем сайте.

Скачать

Скачать самую последнюю версию модуля с репозитория GitHub: abricos-mod-notify.zip

Конфигурация

Все настройки производятся в файле /includes/config.php

Отладка отправляемых сообщений 

$config['module']['notify'] = array(
	"totestfile" => true
);

Если включена отладка отправляемых сообщений, то письма не уходят на почту, а создаются в виде отдельных файлов в папке /cache/eml. При этом папка /cache/eml должно иметь права на запись.

Настройка SMTP 

$config["module"]["notify"] = array(
	"SMTP" => true,                          // Использовать SMTP
	"SMTPHost" => "mail.yourdomain.com",     // SMTP сервер
	"SMTPPort" => 26,                        // SMTP порт
	"SMTPAuth" => true,                      // использовать авторизацию SMTP
	"SMTPUsername" => "yourname@youdomain",  // имя пользователя SMTP
	"SMTPPassword" => "yourpassword"         // пароль пользователя SMTP
);
