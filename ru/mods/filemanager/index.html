Модуль «Менеджер файлов»


Всю работу с файлами и картинками на Вашем сайте берет на себя этот модуль.

  • Персональное файловое хранилище на сайте для каждого пользователя
  • Настройка ролей доступа
  • Настройка квот для определенных групп пользователей
  • Настройка разрешенных типов файлов и их объема
  • Настройка допустимых размеров картинок
  • Автоматическое изменение размера картинок
