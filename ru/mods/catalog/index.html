Модуль «Каталог»

Базовый модуль для построения сложных каталогов. Используется в различных системах, где необходимо динамически строить древовидную структуру из элементов каталога с неопределенным набором полей. Например, Интернет-магазин построен на базе этого модуля.

  • Создание полей элементов каталога в панели управления
  • Древовидная структура разделов
  • Хранение изображений в разделах и элементах каталога

Зависимые модули

Менеджер файлов для хранения изображений в раздела и элементах каталога.
